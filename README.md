# Sales Data Anslysis and Prediction using Python

## Project Overview 
This project analyzes sales data from a small shop to find business insights and predict future sales.

## Technologies used 
-Python 
-pandas - Data manipulation
-numpy - Numerical operations
-matplotlib - Data visualization 
-scikit-learn - Machine learning
